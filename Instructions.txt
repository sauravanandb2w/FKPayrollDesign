Stage1:1) To create Employee Database consisting of 3 tables(master,weekly_table,monthly_table):
________________________________________________________________________________________________________
	->move inside source Directory
	->then run::javac DatabaseConnection.java -d ClassFiles
	->then run::java -cp "ClassFiles;mysql-connector-java-5.1.49-bin.jar" DatabaseConnection
	_________________________________________________________________________
	Result: All three tables are created on mysql server hosted at 127.0.0.1 at port 3306.
